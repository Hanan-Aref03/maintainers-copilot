# Pandas Issue Prompts

These prompts are derived from the repo's pandas issue corpus in `data_pipeline/pandas_issues.json`.

Each file is a prompt card that describes a real issue shape and asks for a short maintainer-oriented triage response.

## Label Coverage

- `bug/` - correctness and performance regressions
- `feature/` - enhancement requests and new APIs
- `docs/` - documentation gaps and clarity problems
- `question/` - open design or behavior questions

## How To Use

- Use these as few-shot examples for triage, prompt tuning, or human review.
- Keep the responses short and practical so they stay useful in a maintainer workflow.
